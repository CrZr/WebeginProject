# WeBegin Project

## Concept

Outil de monitoring sur des bases de données

## Features

- Notifications (alertes sur les bdd)
- Affichage de graphiques
- Intégration graphiques de base de données

## Archi

- Serveur nodeJs
- Bdd mysql
- Front html, TailwindCss
- Potentielle dockerisation

## Tipss
